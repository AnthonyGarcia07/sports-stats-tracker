# Sport Stats Tracker CLI

A command-line sports stats tracker built in Python.
This was my first portfolio project, built to practice
Python, APIs, and Git 

## What it does
- Searches for a player and gets their name, team, sport, and nationality 
- Searches for a team and gets their name, sport, league, and stadium
- Team search is limited to Arsenal due to the free API tier restriction
- Handles invalid searches and lets you search multiple times

## Requirements
- Python 3
- requests library - install it with:
pip install requests

## How to run it
python main.py
